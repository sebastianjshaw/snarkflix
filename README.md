# Snarkflix - Snarky Movie Review Blog

A modern, responsive movie review blog built with HTML, CSS, and JavaScript. Features a snarky tone, AI-generated scores, and a delightful Bernedoodle mascot.

## 🎬 About Snarkflix

At Snarkflix, we are a small but mighty team consisting of a single reviewer and our trusty sidekick, a Bernedoodle. Despite our size, we are passionate about movies and love nothing more than sharing our thoughts and opinions with others. Our unique sense of humor sets us apart from the rest, as we take a lighthearted and entertaining approach to critiquing films.

## ✨ Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Accessibility**: WCAG 2.1 AA compliant with keyboard navigation and screen reader support
- **SEO Optimized**: Structured data, meta tags, and semantic HTML for better search rankings
- **Category Filtering**: Browse reviews by movie genre (Action, Comedy, Drama, etc.)
- **AI Integration**: SnarkAI scores and summaries for each review
- **Modern UI**: Clean, modern design with smooth animations and transitions
- **Performance**: Optimized images, lazy loading, and efficient code

## 🎯 Review Format

Each review follows a consistent structure:

- **Title**: Movie name and year
- **Publish Date**: When the review was published
- **Reading Duration**: Estimated time to read
- **SnarkAI Score**: AI-generated score out of 100
- **AI Review Summary**: TLDR summary generated by SnarkAI
- **Disclaimer**: Information about AI-generated content
- **Header Image**: Eye-catching movie poster or scene
- **Tagline**: Snarky one-liner about the movie
- **Review Content**: Detailed review with personal insights
- **Images**: AI-generated images based on reviewer descriptions
- **YouTube Trailer**: Embedded movie trailer

To add new reviews:

1. **Update JavaScript**: Add review data to the `snarkflixReviews` array in `script.js`
2. **Create Review Page**: Use `review-template.html` as a template
3. **Add Images**: Include movie posters and AI-generated images
4. **Update Categories**: Ensure category counts are accurate

## 🛠️ Customization

### Changing Colors
Update the CSS custom properties in `:root`:

```css
:root {
    --snarkflix-primary: #your-color;
    --snarkflix-secondary: #your-color;
    --snarkflix-accent: #your-color;
}
```

### Adding New Categories
1. Add category data to `snarkflixCategories` array in `script.js`
2. Add corresponding CSS class for category colors
3. Update the categories grid in `index.html`

### Modifying Review Format
Edit the `createReviewElement()` function in `script.js` to change how reviews are displayed.
